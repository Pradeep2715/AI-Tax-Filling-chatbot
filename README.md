# 🤖 TaxBot - AI Powered Tax Filing Assistant

An intelligent AI-powered chatbot that helps users calculate their income tax, compare tax regimes, and receive personalized tax-saving recommendations through a conversational interface.

## 🚀 Features

- 🔐 Secure User Authentication
- 📧 Email OTP Verification
- 💬 AI-Powered Conversational Chatbot
- 📊 Income Tax Calculation
- ⚖️ Old vs New Tax Regime Comparison
- 💡 Personalized Tax Saving Suggestions
- 📁 Chat History Management
- 📱 Responsive Modern UI
- 🌌 Animated Glassmorphism Design
- ⚡ Fast and Interactive Experience

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/taxbot.git
```

### Move into Project

```bash
cd taxbot
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
